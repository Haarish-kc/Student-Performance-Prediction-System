# Student Exam Score Prediction using Machine Learning

## Project Description

This project focuses on predicting student exam scores using Machine Learning and Data Analysis techniques. The goal of the project is to analyze how different academic and lifestyle factors influence a student's final exam performance. By using historical student data, a predictive model is developed that can estimate the expected exam score of a student based on multiple input features.

The dataset used in this project contains more than 6,600 student records with several attributes related to study habits, personal factors, and academic background. Some of the important features in the dataset include hours studied, attendance percentage, sleep hours, previous academic scores, motivation level, internet access, family income, and participation in extracurricular activities. The target variable of the dataset is the **Exam_Score**, which represents the final score obtained by a student.

The project begins with data preprocessing and cleaning to handle missing values and convert categorical variables into numerical format. After preprocessing, exploratory data analysis is performed to understand the relationships between different variables and exam scores. Various visualizations such as correlation heatmaps and scatter plots are used to analyze patterns and trends in the dataset.

Once the data analysis stage is completed, the dataset is divided into training and testing sets using the train-test split technique. Machine Learning regression algorithms are then applied to train the predictive model. The primary algorithm used in this project is **Linear Regression**, which helps predict the exam score based on multiple independent variables. Additional models such as Decision Tree Regressor and Random Forest Regressor can also be used to improve prediction accuracy.

The trained model is evaluated using performance metrics such as Mean Squared Error (MSE) and R² score to measure how well the model predicts exam scores. The results show how data-driven techniques can help understand student performance and identify important factors that influence academic success.

This project demonstrates practical applications of Python libraries such as **Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn** for data analysis and machine learning. It also provides a complete workflow from data preprocessing and visualization to model training and evaluation.

Overall, the project highlights how machine learning can be used to analyze educational data and build predictive systems that support academic performance analysis.
